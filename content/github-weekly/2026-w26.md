### 1. [codex-orange-book](https://github.com/bozhouDev/codex-orange-book)

Codex 橙皮书是一份从安装到实战的全链路 OpenAI Codex 使用指南（非官方开源，含可下载 PDF）。项目涵盖 Codex App、CLI、IDE Extension 和 Web 四种使用方式，包含 5 个实战案例（如构建宠物零食店网站），并详细讲解 automation、plugins、MCP 和 memory 等核心功能。适合 AI 编程新手和希望系统学习 Codex 的开发者。

### 2. [CWR](https://github.com/BohemiaInteractive/CWR)

Bohemia Interactive 开源的 Arma: Cold War Assault Remastered 游戏引擎源代码，是经典军事模拟游戏 Operation Flashpoint 的 25 周年重制版。项目将原始引擎移植到现代 64 位 Windows 和 Linux 平台，支持宽屏显示和更好的硬件兼容性，同时作为官方资产包免费开放。适合游戏开发者、mod 制作者以及对军事模拟引擎感兴趣的程序员学习和二次开发。

### 3. [Qwen-AgentWorld](https://github.com/QwenLM/Qwen-AgentWorld)

Qwen-AgentWorld 是首个覆盖七个 agent 交互域（MCP、Search、Terminal、SWE、Android、Web、OS）的原生语言世界模型。它通过 long chain-of-thought reasoning 模拟 agent 环境，给定 agent 的动作和交互历史即可预测下一个环境状态，提供 35B-A3B 和 397B-A17B 两种规模。适合研究 agent 环境模拟、强化学习以及需要高保真环境建模的 AI 研究者。

- {icon:lucide:star} 489 Star | {icon:lucide:code} Python | {icon:lucide:git-fork} 47 Fork

### 4. [AgentSpace](https://github.com/HKUDS/AgentSpace)

AgentSpace 是香港大学数据科学实验室（HKUDS）开发的 agent-native 协作工作空间，核心理念是"Human + Agents. One Team. One Workspace"。项目支持招募和分配 agent、协调多 agent 工作流，将 AI agent 视为一等团队成员而非单纯工具，提供 Inbox 和 Agents 仪表盘等协作界面。适合需要人机协同完成复杂任务的团队和开发者。

### 5. [wloc](https://github.com/Yu9191/wloc)

### 6. [liquid-glass](https://github.com/samasante/liquid-glass)

这是一个基于 React 的无头(headless)组件，能够在网页上实现真实的 DOM 折射效果，模拟 Apple 风格的 Liquid Glass 视觉体验。与简单的模糊效果不同，它能真正扭曲下方的文本和图像，保持文字可选择、链接可点击。支持 Chrome、Safari 和 Firefox 浏览器，零运行时依赖，适合需要实现高级玻璃拟态 UI 效果的前端开发者使用。
