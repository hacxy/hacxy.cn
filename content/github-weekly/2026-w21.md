## 本周新星

### 1. [FoundZiGu/GuJumpgate](https://github.com/FoundZiGu/GuJumpgate)

一个全自动 GPT Plus 注册浏览器扩展，基于 FlowPilot 实现从账号注册、Stripe 账单填写到 PayPal 支付的全流程自动化。支持 Hotmail 别名管理、PayPal 号码池和 OAuth 回调，适合需要批量管理 ChatGPT Plus 订阅的用户。项目提示当前 OAuth 风控较严，推荐使用 Session JSON 导入模式。

### 2. [thananon/9arm-skills](https://github.com/thananon/9arm-skills)

- {icon:lucide:star} 1.9k | {icon:lucide:code} Shell | {icon:lucide:git-fork} 257

### 3. [perplexityai/bumblebee](https://github.com/perplexityai/bumblebee)

Perplexity 出品的只读供应链风险扫描器，用于检测开发者机器上的包、扩展和工具元数据是否暴露于已知的供应链攻击。单个 Go 静态二进制文件，零依赖，支持 baseline/project/deep 三种扫描级别，读取 lockfile 和包管理器元数据但不执行任何命令，适合安全团队在应急响应中快速排查受影响终端。

- {icon:lucide:star} 1.7k | {icon:lucide:code} Go | {icon:lucide:git-fork} 127

### 4. [Doorman11991/smallcode](https://github.com/Doorman11991/smallcode)

专为 8B-35B 参数的本地小模型优化的终端 AI 编码 agent。与面向前沿大模型的工具不同，SmallCode 通过预算管理的上下文摘要、多格式容错的工具调用解析、TODO 文件分解的规划策略和 search-and-replace 补丁编辑来弥补小模型的不足，完全本地运行无需联网，适合注重隐私的开发者。

- {icon:lucide:star} 1.4k | {icon:lucide:code} JavaScript | {icon:lucide:git-fork} 96

### 5. [sapientinc/HRM-Text](https://github.com/sapientinc/HRM-Text)

基于 HRM（层级循环架构）的 1B 参数文本生成模型，核心亮点是用约 $1000 的计算成本从零预训练一个基础模型——相比传统方案节省 130-600 倍算力和 150-900 倍数据。提供完整的预训练框架，包含 PrefixLM 序列打包、FlashAttention 3 内核和 PyTorch FSDP2 分布式训练，在 GSM8k、MATH 等基准上表现优异。

- {icon:lucide:star} 696 | {icon:lucide:code} Python | {icon:lucide:git-fork} 63

### 6. [kageroumado/phosphene](https://github.com/kageroumado/phosphene)

macOS Tahoe 专属的视频壁纸引擎，通过 Apple 私有的 WallpaperExtensionKit 框架将视频文件设为桌面和锁屏壁纸，与系统原生 Aerials 壁纸无缝集成。支持无间隙循环播放、多显示器独立设置、电源感知的自适应播放策略，以及窗口完全遮挡时自动暂停渲染以节省资源。仅支持 Apple Silicon + macOS 26。
