# 一、Neovim的安装

## 什么是Neovim

官方的定义是：基于 [vim](https://zh.wikipedia.org/wiki/Vim) 的超集

实际上就是基于vim的重构版，它保留了vim的特性和功能，并在vim的基础之上加入了更多特性，如：使用Lua语言编写插件、内置lsp客户端、暴露了更多非常好用的API等; 截止目前（2025.12.26）的稳定发行版版本号是0.11.5


```sh
❯ nvim --version
NVIM v0.11.5
Build type: Release
LuaJIT 2.1.1765228720
Run "nvim -V1 -v" for more info
```
