# 二、Neovim的初次邂逅

打开终端应用程序，并在其中输入 `nvim` 命令并运行，以打开Neovim界面, 默认的欢迎界面应该是以下这样：

![](https://hacxy-1259720482.cos.ap-hongkong.myqcloud.com/images/20260108195538188.png)

## 交互式教程

在打开Neovim之后, 你应该做的第一件事是在Neovim中运行命令`:Tutor`

如果你不知道这是什么意思，请依次输入以下内容：

`<escape键>`

`:`

`Tutor`

`<enter键>`

之后你就会看到:

![](https://hacxy-1259720482.cos.ap-hongkong.myqcloud.com/images/20260108200309433.png)

这个交互式教程是一个非常好的开始，请完整的阅读并按照教程中的指示进行操作，你将在这个教程中学习Neovim最基本也是最常用的操作。

你可以反复的进入交互式教程进行练习，一旦你熟练掌握这些技巧，将极大的提升你编辑文本的效率。

:::tip
如果你已经非常熟悉Neovim的基本操作，请跳过此步骤
:::

## 在Neovim中运行lua代码

按下`:`进入命令行模式，接着输入`lua 需要执行的lua代码`

例如： `lua print('hello world')`, 然后按下<enter键>
