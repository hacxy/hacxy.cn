# 撰写文章

## Frontmatter

每篇文章顶部可以用 `---` 包裹一段 YAML 格式的元数据，所有字段都是可选的：

```yaml
---
title: 文章标题
date: 2026-05-10
tags:
  - TypeScript
  - React
summary: 这段话会以 AI 摘要的形式展示在文章开头
sort: 1
---
```

### 各字段说明

**title** — 文章标题。如果不写，会自动取正文中的第一个 `# 标题`；如果正文也没有标题，就用文件名。

**date** — 发布日期。如果不写，会自动读取这个文件最后一次 git 提交的日期。

**tags** — 标签列表。会展示在文章详情页的标题下方，也用于标签页的分类筛选。

**summary** — AI 摘要文本。设置后文章详情页会出现一个「AI 摘要」组件，点击后以打字机效果逐字展示摘要内容。

**sort** — 排序权重。用于控制文章在侧边栏中的位置，数值越小越靠前。不设置则按日期倒序排列。

## 写一篇最简单的文章

其实什么 frontmatter 都不写也行：

```markdown
# 我的文章

直接开始写内容。
```

标题取自 `# 我的文章`，日期取自 git 提交时间，没有标签，没有摘要。一切自动。
