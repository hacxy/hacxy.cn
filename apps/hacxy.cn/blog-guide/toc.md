# 文章目录

每篇文章的右侧会自动展示一个目录（Table of Contents），从文章内容中提取 h2 到 h4 级别的标题。

## 自动生成

不需要任何配置。只要文章里有二级、三级或四级标题，右侧就会出现目录。

```markdown
## 这是二级标题     ← 会出现在目录里
### 这是三级标题    ← 会出现在目录里，缩进一级
#### 这是四级标题   ← 会出现在目录里，缩进两级
##### 这是五级标题  ← 不会出现
```

## 滚动高亮

滚动页面时，目录会自动高亮当前可见的标题。

## 点击跳转

点击目录中的任意标题，页面会平滑滚动到对应位置。跳转时会自动预留顶部导航栏的高度，标题不会被遮挡。

## 响应式

在小于 1200px 的屏幕上，目录会自动隐藏。
